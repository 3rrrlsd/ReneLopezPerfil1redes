Componentes de Red y Cableado Estructurado

1. ¿Qué son los componentes pasivos en la Red?

Los componentes pasivos son aquellos elementos de una red que no requieren de energía eléctrica para funcionar y que simplemente se encargan de transportar o canalizar la señal de red. Ejemplos de componentes pasivos incluyen cables, conectores, canaletas, paneles de conexión y más.

2. Descripción técnica de cada componente pasivo con su respectiva imagen

RJ45:

El conector RJ45 es un tipo de conector modular utilizado para conectar cables de red en sistemas Ethernet. Tiene 8 pines que permiten la conexión de los hilos de un cable UTP.

Jacks o Keystone:

Los Jacks o Keystone son módulos que se montan en placas para conectar los cables de red. Se utilizan en puntos de red y paneles de parcheo.

Cable UTP:

El cable UTP (Unshielded Twisted Pair) es un cable de par trenzado no blindado usado comúnmente en redes de datos y telecomunicaciones.

3. ¿Qué son los componentes activos en la Red?

Los componentes activos son aquellos dispositivos en una red que requieren energía eléctrica para funcionar. Se encargan de gestionar, amplificar y redirigir las señales de la red.

4. Descripción técnica de cada componente activo

Switch:

El switch permite interconectar varios equipos dentro de una red, facilitando la comunicación entre ellos. Recibe y envía datos solo al puerto correspondiente.

Router:

El router es un dispositivo que permite la comunicación entre diferentes redes, dirigiendo los paquetes de datos hacia su destino correcto.

5. Herramientas utilizadas para realizar cableado estructurado

Ponchadora:

La ponchadora es una herramienta utilizada para insertar los cables en los conectores RJ45.

6. ¿Qué es el cableado horizontal?

El cableado horizontal conecta las estaciones de trabajo a los puntos de conexión en un área específica, como un piso de un edificio.

7. ¿Qué es el cableado vertical o backbone?

El cableado vertical o backbone conecta los cuartos de telecomunicaciones y equipos principales de los diferentes pisos de un edificio.

8. Tipos de cables para el sistema de cableado horizontal

Cable UTP Categoría 5e:

Este cable soporta velocidades de hasta 1000 Mbps (Gigabit Ethernet). Es común en instalaciones residenciales y pequeñas oficinas.

10. ¿Para qué se utiliza un cable directo en la red?

El cable directo se utiliza para conectar dispositivos diferentes, como una computadora a un switch o un router.

11. ¿Para qué se utiliza un cable cruzado en la red?

El cable cruzado se utiliza para conectar dispositivos similares, como dos computadoras o dos switches.
